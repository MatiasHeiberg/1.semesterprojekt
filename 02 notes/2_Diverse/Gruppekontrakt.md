**Hvad gør vi ved hvis vi ikke kan mødes?**
- Discord til kommunikation. 

**Hvordan pacer vi vores projekt?**
- Bred implementering først, dybde senere hvis tiden tillader det.
- Horisontalt fokus frem for vertikal.

**Hvad gør vi hvis vi bliver presset på tiden?**
- Vi kan lave to slags kompris; (1) begrænse scope/kvalitet og (2) uddeleger opgaverne og gå på kompromis med resten af holdets indsigt i arbejdet.
- (1) Er vi mest komfortabel med hvis det er "stuearbejde" vi er presset på.
- (2) er vi mest komfortabel med hvis det er "implementeringsarbejde".

**Hvad gør vi med "lektier"?**
- Enighed om at vi gerne vil lave noget der hjemme.
- Samler op på det individuelle arbejde hver morgen.
- Det er *nice to have, not need to have*.
- Tager ikke nogen designbeslutninger der hjemme selv.

**Hvordan undgår vi at bruge for meget tid i OOA og problemdomænet?**
- Undgå perfektionisme
- Fokuser på at godt, er godt nok.
- Fokuser på at godt nok er bedre end perfekt.

**Starter vi dagen med et morgenmøde?**
- Samler op fra dagen før og forbinder dagens tankestrøm med hvor vi slap.
