På baggrund af vores 'must'-krav i MoSCoW-modellen har vi udarbejdet Use Cases, der opfylder minimumskravene for projektet og uddybes i Casual-form og bliver mere case-specifikke. De tre nedenstående har været vores udgangspunkt i hele design- og implementeringsfasen og vil herefter refereres til som Use case 2, 3 eller 4. Se bilag for resterende Use Cases i bilag (INDSÆT BILAG)
